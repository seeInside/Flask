## Flask
* 这是我在学习Flask Web的过程中，写的一个博客项目，参考书籍《Flask Web开发——基于Python的Web应用开发实战》，目前对整个框架有了一个基本大概的了解，也深知还有非常多的知识需要学习与总结。
# 简介
* 使用的模板： Twitter Bootstrap
* 数据库： SQLite
* 可作为Web Service向移动端提供API接口
